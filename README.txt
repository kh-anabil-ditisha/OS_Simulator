Mini Operating System Simulator

Selected language from assignment sheet: Python
Framework: Flask
Frontend: HTML, CSS, JavaScript

How to run:
1. Install Python
2. Open this folder in VS Code or terminal
3. Run: pip install flask
4. Run: python app.py
5. Open browser: http://127.0.0.1:5000

Implemented easy 3 modules:
1. CPU Scheduling Simulator
2. Page Replacement Simulator
3. Deadlock Handling using Banker's Algorithm
